// Transfer Function

Output voltage (ideal)
V_OUT = V_REF × (D / 2^N)

D = D3×8 + D2×4 + D1×2 + D0×1
N = 4 bits → range 0..15
Example codes
1111 → V_REF × 15/16 = 0.9375 V_REF
1000 → V_REF × 8/16 = 0.5000 V_REF
0001 → V_REF × 1/16 = 0.0625 V_REF
0000 → 0V
Thevenin output impedance
Z_OUT = R (code-independent)

← This is the key property.
Constant regardless of D code.
MSB high-current trick
Use low R values (e.g. 100Ω/200Ω)
+ high V_REF (12–24V)
→ large I_OUT, no buffer needed
+ parallel N modules: Z_OUT / N
Switch operation: each SPDT connects the 2R shunt resistor either to V_REF (bit=1) or GND (bit=0). The Thevenin equivalent seen at V_OUT is always R regardless of the code word — this is the fundamental property that makes R-2R work. In MSB's implementation the switches are replaced by current-steering differential pairs (bipolar or FET), keeping the current source always active and only changing its destination — eliminating switching glitches at the output.
